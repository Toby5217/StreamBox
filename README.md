# StreamBox
StreamBox is a hardware and software system that transforms ordinary buttons into a modern computer control tool. Connect a homemade Arduino board with 10 buttons, assign them any actions, and control media, hotkeys, and programs—directly from a beautiful desktop application.
1. Необходимые компоненты
Плата Arduino (Uno, Nano или аналогичная) с USB-кабелем.

10 тактовых кнопок.

Макетная плата и соединительные провода.

Компьютер с установленными:

Arduino IDE (для загрузки скетча).

Python 3.6+ и библиотека pyserial (установка: pip install pyserial).

2. Схема подключения
Каждая кнопка подключается одним контактом к цифровому пину Arduino (пины 2–11), вторым — к GND.

Внутренние подтягивающие резисторы активируются программно, внешние резисторы не требуются.

Рекомендуется подписать кнопки на макетной плате номерами от 1 до 10.

3. Настройка и запуск Arduino
Откройте Arduino IDE, скопируйте приведённый выше скетч.

Выберите правильную плату и порт в меню "Инструменты".

Загрузите скетч на плату.

Откройте "Монитор порта" (9600 бод) для проверки: при нажатии кнопок должны появляться строки вида 3:0, 3:1.

4. Настройка и запуск Python-моста
Убедитесь, что Arduino подключена к USB.

В коде Python переменная SERIAL_PORT изначально равна None — программа попытается найти Arduino автоматически. Если автоматика не сработала, укажите порт вручную, например SERIAL_PORT = "COM3" (Windows) или SERIAL_PORT = "/dev/ttyUSB0" (Linux).

Запустите скрипт:

bash
python bridge.py
При нажатии кнопок в консоли будут появляться сообщения с временем и событием. Все события записываются в файл button_log.csv.

5. Возможные проблемы и их решение
Порт не найден: Проверьте, установлен ли драйвер USB-UART (для Arduino с CH340 может потребоваться драйвер). Укажите порт вручную.

Ошибка доступа к порту: Закройте другие программы, использующие порт (например, монитор порта Arduino IDE).

Мусор в данных: Убедитесь, что скорость в скетче и Python совпадают (9600). Проверьте подключение кнопок (нет ли замыканий).

6. Расширение функциональности
Проект спроектирован с возможностью лёгкого расширения:

Для эмуляции клавиатуры используйте библиотеку pyautogui и добавьте вызовы в блок обработки событий.

Для графического интерфейса можно использовать tkinter и обновлять виджеты при получении данных.

Для сетевых функций (MQTT, HTTP) добавьте соответствующие вызовы в то же место.
